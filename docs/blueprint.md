# **App Name**: EGX Portfolio Tracker

## Core Features:

- Investment Input: Allow users to input investment amounts, purchase date, and asset type (Egyptian stocks or gold).
- Price Tracking: Connect to a low-cost/free open API for EGX stock and gold prices to automatically fetch current values.
- Performance Calculation: Automatically calculate investment performance (earnings) for daily, monthly, and yearly periods based on purchase price and current value.
- Searchable Tracker: Implement a searchable table for tracking the prices of available stocks and Gold in the egyptian stock market.
- AI Investment Summary: Use an AI tool to provide a short, personalized summary of the user's overall investment performance each month, including key gains and losses, using available information.
- Data Storage: Save investment data with Firestore.

## Style Guidelines:

- Primary color: A vibrant green (#3dd598) evoking growth and investment success.
- Background color: A light gray (#f7f7f7) provides a clean, modern backdrop, contrasting with the primary color.
- Accent color: A light blue (#9ad3e6) highlights key interactive elements and data points.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will offer a modern, neutral look for both headlines and body text.
- Use clear, minimalist icons to represent different investment types and performance metrics.
- Clean and intuitive layout inspired by Robinhood, prioritizing easy access to key data and performance metrics.
- Subtle animations for loading data and displaying performance updates, providing a smooth user experience.
